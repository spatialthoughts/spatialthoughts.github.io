# Resources

A collection of free and open learning resources I have built to help people master modern geospatial technologies.

<div class="grid" markdown>

<div class="project-card" markdown>

**[SpatialThoughts OpenCourseWare](https://courses.spatialthoughts.com/)**

A collection of free, self-paced courses covering QGIS, Python, Google Earth Engine, and more. All course materials are open and freely available.

[Visit Site](https://courses.spatialthoughts.com/){ .md-button }
</div>

<div class="project-card" markdown>

**[QGIS Tutorials and Tips](https://www.qgistutorials.com/)**

A comprehensive tutorial site with step-by-step guides for QGIS. One of the most widely used QGIS learning resources on the web.

[Visit Site](https://www.qgistutorials.com/){ .md-button }
</div>

<div class="project-card" markdown>

**[GeoPython Tutorials](https://www.geopythontutorials.com/)**

Tutorials focused on geospatial data analysis using Python, covering libraries like GeoPandas, Xarray, Dask, STAC and more.

[Visit Site](https://www.geopythontutorials.com/){ .md-button }
</div>


<div class="project-card" markdown>

**[Technical Blog Posts](https://spatialthoughts.com/articles/)**

In-depth articles on geospatial analysis, Python, QGIS, GDAL, Mapshaper and more — covering techniques, workflows, and tools for practitioners.

[Read Articles](https://spatialthoughts.com/articles/){ .md-button }
</div>

<div class="project-card" markdown>

**[Spatial Thoughts YouTube](https://www.youtube.com/@SpatialThoughts)**

Video tutorials and course walkthroughs covering a wide range of geospatial topics, from beginner QGIS to advanced Python and Earth Engine.

[Visit Channel](https://www.youtube.com/@SpatialThoughts){ .md-button }
</div>

A free, open textbook for learning Google Earth Engine. I contributed 4 chapters and produced video walkthroughs for the book's exercises.
